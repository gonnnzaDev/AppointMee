# AppointMee

Red Social de Turnos desarrollada con **Spring Boot**.
El proyecto incluye autenticación de usuarios, manejo de roles y sistema de pagos integrado con Mercado Pago.

## Backend

* Java 21
* Spring Boot
* Spring Security
* Spring Data JPA
* Mercado Pago SDK
* Maven
  
## Frontend

* HTML5
* CSS3
* JavaScript
* Tailwind
* Bootstrap
  
## Seguridad

* Spring Security
* Autenticación basada en sesiones
* Control de acceso por roles

## Base De datos
* PostgreSQL (SupaBase)
